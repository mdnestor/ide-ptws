Source code for the paper "Periodic Traveling Waves in an Integro-Difference Equation With a Nonmonotone Growth Function and Strong Allee Effect" by [M. Nestor](https://mdnestor.github.io/) and [B. Li](http://www.math.louisville.edu/~bli/).

# Installation

In terminal:
```sh
git clone https://github.com/mdnestor/idesim.git
cd idesim
```

The module requires numpy and matplotlib. The primary class can be imported as follows:

```python
from idesim import IDESim
```
